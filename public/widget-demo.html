How to Use the Widget

1. Include the Script

<script src="https://yoursite.com/finnrick-rating-widget.js"></script>

2. Use the Custom Element

<finnrick-rating-widget
    company-name="Shanghai Innovy Chemical New Materials"
    products="Tesamorelin, CJC-1295, Ipamorelin"
    tested-samples="6"
    last-test-date="19 Feb 2025"
    grade-text="GREAT"
    grade-color="#0E8A4A">
</finnrick-rating-widget>

3. Available Attributes

  • company-name: The name of the company
  • products: Product information/names
  • tested-samples: Number of samples tested
  • last-test-date: Date of the last test
  • grade-text: Grade text (e.g., "GREAT", "GOOD", "POOR")
  • grade-color: Background color for the grade section (hex color)

4. Dynamic Updates with JavaScript

// Get the widget element
const widget = document.querySelector('finnrick-rating-widget');

// Update attributes dynamically
widget.setAttribute('company-name', 'New Company Name');
widget.setAttribute('grade-text', 'EXCELLENT');
widget.setAttribute('grade-color', '#00AA00');
