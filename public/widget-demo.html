How to Use the Widget

1. Include the Script

<script src="https://yoursite.com/finnrick-rating-widget.js"></script>

2. Use the Custom Element

<finnrick-rating-widget
    company-name="Shanghai Innovy Chemical New Materials"
    products="Tesamorelin, CJC-1295, Ipamorelin"
    tested-samples="6"
    last-test-date="19 Feb 2025"
    grade-text="GREAT"
    grade-color="#0E8A4A">
</finnrick-rating-widget>

3. Available Attributes

  • company-name: The name of the company
  • products: Product information/names
  • tested-samples: Number of samples tested
  • last-test-date: Date of the last test
  • grade-text: Grade text (e.g., "GREAT", "GOOD", "POOR")
  • grade-color: Background color for the grade section (hex color)

4. Test Widget on Any Website (Browser Console)

Want to test the widget on any website instantly? Follow these steps:

  1. Open any website in your browser
  2. Press F12 or right-click and select "Inspect"
  3. Go to the Console tab
  4. Copy and paste this code, then press Enter:
// Load the Finnrick Rating Widget
const script = document.createElement('script');
script.src = 'https://finnrick-cards-test.vercel.app/finnrick-rating-widget.js';
document.head.appendChild(script);

// Wait for script to load, then add the widget
script.onload = () => {
  const widget = document.createElement('finnrick-rating-widget');
  widget.setAttribute('company-name', 'Test Company');
  widget.setAttribute('products', 'Premium Research Compounds');
  widget.setAttribute('tested-samples', '5');
  widget.setAttribute('last-test-date', '19 Jul 2025');
  widget.setAttribute('grade-text', 'EXCELLENT');
  widget.setAttribute('grade-color', '#0E8A4A');
  
  // Add the widget to the top of the page
  document.body.insertBefore(widget, document.body.firstChild);
  
  widget.style.cssText = `
    position: fixed;
    top: 20px;
    right: 20px;
    z-index: 9999;
    box-shadow: 0 4px 20px rgba(0,0,0,0.3);
  `;
};
